# Outputs

The orchestrator writes all deliverables to this folder.

## Expected Output Files

| File | Produced By | Description |
|------|------------|-------------|
| `risk-summary.md` | Research Synthesizer Agent | Ranked list of auditable areas with risk scores, rationale, and prior finding history |
| `standards-alignment.md` | Standards Alignment Agent | Mapping of scope items to IIA Standards and control frameworks, with coverage gap analysis |
| `loe-estimate.md` | LOE Estimator Agent | Hours by phase and staff level, timeline, resource allocation, and assumptions |
| `scoping-package.md` | Orchestrator | The final combined deliverable with executive summary, risk-based scope, standards alignment, LOE estimate, exclusions, and approval placeholders |

## Important

All outputs are drafts. Every conclusion and recommendation is marked for auditor review. The scoping package is an input to your planning process, not a replacement for professional judgment.

Review the scoping package with your engagement team before presenting to leadership or the board.
